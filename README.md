# Mini Quiz

This is a simple Python quiz game for beginners. It asks 10 multiple-choice math questions, checks your answers, and shows your score at the end. The correct answers are **randomized**, so the correct option is not always in the same position.  

## How to Run
1. Make sure you have Python installed on your computer.
2. Open terminal or command prompt.
3. Navigate to the folder containing `miniquiz.py`.
4. Run the program with:
5. Follow the instructions and answer the questions by typing the number of the option you think is correct.  

## Features
- 10 multiple-choice math questions
- Randomized answer positions
- Tracks your score and gives feedback at the end
